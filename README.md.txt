# Telecom X – Parte 2: Prevendo Churn

📊 **Desafio de Machine Learning** para prever evasão de clientes na Telecom X.  
Este projeto faz parte da continuidade da análise exploratória (Parte 1), agora focado em **modelagem preditiva**.

---

## 🎯 Objetivos
- Pré-processamento de dados (tratamento, encoding, normalização).
- Análise de correlação e seleção de variáveis.
- Construção de modelos preditivos de classificação.
- Avaliação do desempenho dos modelos.
- Interpretação dos resultados e extração de insights estratégicos.
